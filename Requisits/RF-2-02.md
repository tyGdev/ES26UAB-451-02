# Requirement
### ID
RF-2-02

### Type / Category
Functional

### Version
1.0

### Title
Vista per plats i menús

### Description
L'usuari pot visualitzar un llistat de plats i menús individuals disponibles en el moment actual.

### Relations
[RF-2-01](./RF-2-01.md)
[RF-2-03](./RF-2-03.md)
[RF-2-04](./RF-2-04.md)

###Coments
La vista de platos son como indica el nombre platos unicos con descripcion del mismo plato mas descripcion breve del cocinero (raiting, ubicación). Por otra parte el menu se enseñara segun el nombre del menu mas una imagen descriptiva de lo entra dentro del menu, la descripcion del menu entra una descripcion general + cocinero como en platos, seguido de los platos que entran dentro del menu, si estos són platos individuales ya existentes se creara un link en el nombre del plato que abrira una nueva pagina para ver correctamente en detalle las especificaciones del mismo plato. Por lo contrario si es un plato exclusivo de menu, se habilitara despues del nombre del plato la descripcion del mismo mas un remarcante extra para que se entinda que es unico al menu. Por la parte del precio aparte del precio del menu se vera el precio por platos individuales mas el precio de los adicionales del menu como pueden ser bebidas o postre.
