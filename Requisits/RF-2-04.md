# Requirement
### ID
RF-2-04

### Type / Category
Functional

### Version
1.1

### Title
Ordenació per distància

### Description
Ambdues vistes (plats i cuiners) mostren els resultats ordenats per distància física entre la ubicació del client i la del cuiner.

### Relations
[RF-2-02](./RF-2-02.md)
[RF-2-03](./RF-2-03.md)

### Coments:
Com adicional al la ordenació es mostrara primer millors raitings per cuiner per distancies que coincideixin en un rang del 5%
