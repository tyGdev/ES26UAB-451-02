# Requirement
### ID
RF-2-01

### Type / Category
Functional

### Version
1.0

### Title
Llistat de plats filtrat per restriccions

### Description
El sistema mostra primer els plats que compleixen les restriccions alimentàries indicades en el perfil del client. Els plats que no compleixen les restriccions poden seguir apareixent al llistat.

### Relations
[RF-1-04](./RF-1-04.md)
[RF-2-02](./RF-2-02.md)
[RF-2-03](./RF-2-03.md)

### Comments
Para que no sea exclusivo la vista de alimentos que cumplen las restricciones alimenticias se marcara claramente donde comienzan platos o menus que no cumples estas restricciones al final de la pagina, estas restricciones son las previamente guardadas en perfil o se podra añadir adicinalmente como filtro despues de una busqueda
