# Requirement
### ID
RF-2-06

### Type / Category
Functional

### Version
1.1

### Title
Reserva de comanda futura

### Description
Si un cuiner està marcat com a no disponible, el client pot reservar una comanda per a l'hora en què el cuiner estigui disponible.

### Relations
[RF-2-03](./RF-2-03)
[RF-2-09](./RF-2-03)

### Comments:
El cuiner té un límit de comandes que pot preparar segons el temps que triga a cuinar i no es poden fer reserves arribat aquest límit.
